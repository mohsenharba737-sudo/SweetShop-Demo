# نموذج واجهة ويب ديناميكية لمتجر إلكتروني

هذا المشروع هو **نموذج تعليمي تفاعلي** لتطوير واجهات ويب ديناميكية ومتجاوبة باستخدام **HTML, CSS, وJavaScript**. يركز على تعلم التفاعل مع DOM، إدارة سلة المشتريات، التبديل بين اللغات، وعرض المنتجات بطريقة Slider.

## ⭐ المميزات الرئيسية

- **واجهة متجاوبة بالكامل**: تعمل على جميع أحجام الشاشات.
- **تبديل اللغة**: دعم العربية (RTL) والإنجليزية (LTR) ديناميكيًا.
- **سلة مشتريات ديناميكية**:
  - إضافة وحذف المنتجات والعروض.
  - حساب المجموع الكلي تلقائيًا.
  - إتمام الطلب عبر WhatsApp.
- **شريط تنقل ثابت**: للتنقل السلس بين الأقسام.
- **Slider أفقي للمنتجات والعروض** مع أزرار تمرير يسار/يمين.
- **تخزين مؤقت للسلة** باستخدام `localStorage` للحفاظ على المنتجات بعد إعادة تحميل الصفحة.

---

## 🛠 التقنيات المستخدمة

- **HTML5**: لبناء الهيكلية والأقسام المختلفة.
- **CSS3**: تصميم متجاوب، تأثيرات hover، ألوان جذابة باستخدام Flexbox وBox-shadow.
- **JavaScript (Vanilla JS)**:
  - إدارة المنتجات والعروض.
  - تحديث السلة وإجمالي الأسعار.
  - التبديل بين اللغات وRTL/LTR ديناميكيًا.
  - التمرير السلس للأقسام والمنتجات.

## ⚡ كيفية الاستخدام

1. افتح الملف `index.html` في المتصفح مباشرة.
2. استخدم شريط التنقل للتنقل بين الأقسام.
3. أضف منتجات أو عروض إلى السلة باستخدام زر "أضف للسلة".
4. اضغط على أيقونة السلة لعرض محتوياتها وحذف أي عنصر.
5. اضغط على زر "إتمام الطلب عبر واتساب" لإرسال الطلب مع تفاصيل المنتجات والمجموع الكلي.
6. استخدم زر اللغة للتبديل بين العربية والإنجليزية.

## 🎯 الهدف من المشروع

- نموذج عملي لتعلم:
  - تصميم واجهات ويب متجاوبة وسريعة الاستجابة.
  - إدارة محتوى المنتجات ديناميكيًا باستخدام JavaScript.
  - التفاعل مع DOM لتحديث النصوص والمحتوى.
  - استخدام `localStorage` لتخزين بيانات السلة مؤقتًا.
  - إنشاء تجربة مستخدم سلسة مع التمرير السلس والتبديل بين اللغات.

## 💡 ملاحظات
